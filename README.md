# SoloAPP

A simple mobile application that allows you to create your own training plan, earn XP and level up for self-motivation

## Features

* **Dynamic Exercise Creation**: Add,edit and delete your exercise in one click
* **RPG**: Earn experience and upgrade your level
* **Mobile Ready**: You can download and use app on your phone

## Installation

### Option 1: Download APK

Download the last APK file from the **Release** section.

### Option 2: Run from Source
1. **Clone the repository**
    ```bash
    git clone https://github.com/asvta/SoloApp.git
    cd SoloAPP

1. **Requirements**:
    python3, kivy, pillow

2. **Run the main.py**:
    ```bash
    python main.py
